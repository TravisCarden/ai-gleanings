# Development Container Configuration

Exemplary devcontainer setup for AI tooling development projects.

## Purpose

This devcontainer demonstrates best practices for:
- Consistent development environment across team members
- Pre-installed AI development tools and dependencies
- Proper volume mounting and workspace configuration
- Integration with VS Code and GitHub Codespaces

## Usage

This configuration can be copied to other projects and adapted as needed. The setup includes common tools for AI development while remaining lightweight and fast to build.

## Philosophy

The devcontainer follows the **80% use case** principle - includes tools most AI development projects need while avoiding bloat from specialized requirements.
