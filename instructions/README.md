# Agent Instructions

Agent-agnostic instruction templates using hybrid approach (principles + concrete examples).

## Philosophy

These instructions work across AI agent platforms by focusing on:
- **Universal principles** rather than platform-specific syntax
- **Concrete examples** that illustrate the principles in action
- **Clear structure** that adapts to different agent contexts

## Structure

Instructions are organized by use case:

```
code-review/         # Code review instruction patterns
project-analysis/    # Codebase and architecture analysis  
workflow-automation/ # Development workflow instructions
```

## Quality Standards

Instructions are tested for:
- **Token efficiency** - Concise while complete
- **Design principles** - Progressive disclosure, clear structure
- **Effectiveness** - Produce good results consistently

## Usage

These are templates to adapt rather than use verbatim. Each instruction includes:
- **Context** - When and why to use it
- **Principles** - Core guidelines that apply across situations  
- **Examples** - Concrete applications of the principles
