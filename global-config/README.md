# Global Configuration Templates

Exemplary templates for user-level AI tooling configuration. These demonstrate patterns and best practices without personal details.

## Structure

- `claude-code/` - Claude Code user settings and keybindings
- `rtk/` - RTK (Rust Token Killer) configuration  
- `shell-integration/` - Shell hooks and functions

## Usage

These are templates, not drop-in replacements. Review and adapt them for your specific setup:

```bash
# Example: Using Claude Code settings template
cp global-config/claude-code/settings.json ~/.claude/settings.json
# Then edit ~/.claude/settings.json for your needs
```

## Philosophy

Templates show the **80% use case** - production-ready patterns that work for most users and can be adapted for specific needs.
